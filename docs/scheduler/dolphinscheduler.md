# DolphinScheduler

## 1. 它是什么？

DolphinScheduler 是用于数据任务编排、调度和运维的工作流系统。

## 2. 它解决什么问题？

它解决 Spark、Shell、SQL、质量校验等任务之间的依赖和周期运行问题。

## 3. 它在整个流程中的位置？

它通常负责调度离线加工、湖仓建模、数据同步和质量校验任务。

## 4. 底层原理是什么？

通过工作流 DAG 描述任务依赖，通过调度器触发运行并记录状态。

## 5. 典型使用场景

每日指标产出、分层任务编排、失败重跑、补数。

## 6. 常见问题

依赖漏配、重跑范围不清、失败告警未触达负责人。

## 7. 优化方案

统一命名、设置任务负责人、规范补数流程和质量节点。

## 8. 和其他技术的区别

DolphinScheduler 不直接提升 SQL 性能，但能保障链路稳定运行。
