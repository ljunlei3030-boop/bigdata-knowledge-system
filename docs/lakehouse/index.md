# 湖仓一体概览

## 1. 它是什么？

湖仓一体是把数据湖的低成本、开放存储与数仓的表管理、计算和治理能力结合起来的数据平台架构。

## 2. 它解决什么问题？

它解决数据多份复制、实时离线割裂、元数据分散和治理难闭环的问题。

## 3. 它在整个流程中的位置？

它连接采集、入湖、计算、建模、查询和治理。

## 4. 底层原理是什么？

通过统一表格式、统一元数据、存算分离和多引擎读写，让数据在湖中沉淀并被不同计算引擎使用。

## 5. 典型使用场景

CDC 入湖、流批一体加工、多引擎查询、数据资产统一治理。

## 6. 常见问题

表格式选择、元数据一致性、文件合并、查询性能和引擎兼容。

## 7. 优化方案

选择合适表格式，规划分层模型，控制文件大小，建设治理规则。

## 8. 和其他技术的区别

湖仓一体不是单个引擎，而是由存储、表格式、计算、查询和治理组成的架构。
