# 学习指南概览

## 1. 它是什么？

学习指南用于说明这个知识库如何组织大数据技术内容，以及如何从单个问题逐步建立系统化理解。

## 2. 它解决什么问题？

它解决零散学习难以复用的问题，把 Spark、Flink、Paimon、ClickHouse、建模、治理和优化放到同一张知识地图中。

## 3. 它在整个流程中的位置？

学习指南是进入知识库的入口，帮助先确定学习路径，再进入具体技术专题、问题库和实战案例。

## 4. 底层原理是什么？

每个主题都按照“是什么、为什么、怎么做、适用场景、注意事项、总结输出”组织，并通过关联知识把页面串联起来。

## 5. 典型使用场景

- 从 Spark 执行流程开始理解性能问题。
- 从湖仓一体链路理解 Paimon 和多引擎计算。
- 从数据建模与治理理解表、指标和质量。

## 6. 常见问题

不要只记录结论，应补充发生场景、排查方式、方案代价和关联知识。

## 7. 优化方案

新增文章时优先补齐结构，再逐步增加图、表、案例和参数说明。

## 8. 和其他技术的区别

这个知识库不是普通文章列表，而是面向长期沉淀的结构化技术地图。

学习指南的作用是让每篇文章都能回到完整技术链路中，形成可持续维护的个人知识体系。
